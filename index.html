Jèrriais.Tech

We are an organisation using technology to promote Jèrriais, the native language of the island of Jersey.

Projects

Jèrriais keyboard

A smartphone keyboard for typing in Jèrriais. Features spelling suggestions courtesy of Geraint Jennings' spell checker file. See the video tutorials for help installing and using the keyboard.

Install the keyboard

Jèrridle

A Jèrriais word game. You have 6 chances to guess the 5-letter Jèrriais word, with clues telling you which letters are correct. Wordle for Jèrriais.

Play Jèrridle

Jèrriais/English dictionary

A digital version of the English/Jèrriais and Jèrriais/English dictionaries. Copyright for the dictionary data is held by Société Jèrsiaise, Jersey Heritage and Le Don Balleine.
